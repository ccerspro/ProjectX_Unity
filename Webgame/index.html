The Room 408 is a psychological observation game inspired by The Exit 8 by KOTAKE CREATE.

Built in Unity, this game places the player in an endless loop of nearly identical corridors. The journey begins in Room 400. The first two corridor segments are always normal, allowing players to learn what "normal" looks like.

After that, players must observe each corridor carefully. If something feels wrong, the correct move is to turn back immediately. If the corridor seems normal, continue forward.

Making 8 correct observations in a row leads to Room 408, where the game can be completed. A single mistake resets the player to Room 400.

This game is designed for desktop browsers. Press ESC to exit the game.

You can also view the source code on GitHub.
